# Selection Sort

## array = [22,27,16,2,18,6] 

### case : 18

- 22 < 27 so;
    * 1 - array = [22,27,16,2,18,6]

- 22 > 16 so; 
    * 2 - array = [22,16,27,2,18,6]
    * 3 - array = [16,22,27,2,18,6] 

- 16 > 2 so;
    * 4 - [16,22,2,27,18,6]
    * 5 - [16,2,22,27,18,6]
    * 6 - [2,16,22,27,18,6]

- 22 > 18 so;           
    * 7 - [2,16,22,18,27,6]
    * 8 - [2,16,18,22,27,6]

- 16 > 6 so;
    * 9 - [2,16,18,22,6,27]
    * 10 - [2,16,18,6,22,27]
    * 11 - [2,16,6,18,22,27]
    * 12 - [2,6,16,18,22,27]

After the array is sorted, it enters the 1st state(Avarage case)
because the number 18 is in the middle.
